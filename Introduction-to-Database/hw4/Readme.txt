Readme

- 처음에 Problem number(integer, 1~5)를 받습니다.
- 이후 문제에서 요구하는 순서대로 user input를 한 줄 씩 받습니다. 어떤 input을 요구하는지는 출력해줍니다.
- table은 header 위치에 column label를 출력합니다.

예시

problem #(int): 1
maximum price: 3673
minimum speed: 1.60
minimum RAM: 1024
minimum hard disk: 60
minimum screen size: 13.3
